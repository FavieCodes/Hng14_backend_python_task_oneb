🚀 Profile Management API

HNG Cohort 14 - Stage 1 Backend Task

📡 REST API | 🔐 No Auth Required | 🌐 CORS Enabled

📋 Overview

This API integrates three external services (Genderize, Agify, and Nationalize) to predict gender, age, and nationality based on a person's name. All data is stored in a database with UUID v7 identifiers and UTC timestamps.

📡 API Endpoints Documentation

POST /api/profiles
Create a new profile - Fetches data from external APIs and stores it. If the name already exists, returns the existing profile.
Request Body:
{
  "name": "ella"  required
}
Response (201 Created):
{
  "status": "success",
  "data": {
    "id": "e84b197e-94c9-4bb5-b8c4-26d19dab6201",
    "name": "ella",
    "gender": "female",
    "gender_probability": 0.99,
    "sample_size": 97517,
    "age": 53,
    "age_group": "adult",
    "country_id": "CM",
    "country_probability": 0.0968,
    "created_at": "2026-04-16T11:04:30.200197Z"
  }
}
GET /api/profiles
Get all profiles - Returns a list of all profiles with optional filtering.
Query Parameters (all optional):
gender - Filter by gender (male/female) - case insensitive
country_id - Filter by country code (NG, US, etc.) - case insensitive
age_group - Filter by age group (child/teenager/adult/senior) - case insensitive
Response (200 OK):
{
  "status": "success",
  "count": 2,
  "data": [
    {
      "id": "uuid-1",
      "name": "emmanuel",
      "gender": "male",
      "age": 25,
      "age_group": "adult",
      "country_id": "NG"
    }
  ]
}
GET /api/profiles/{id}
Get a single profile - Retrieves a specific profile by its UUID.
DELETE /api/profiles/{id}
Delete a profile - Permanently removes a profile from the database.
Response: 204 No Content
